# 🔳 QR Code Generator

A simple and modern QR Code Generator built with HTML, CSS, and JavaScript.  
Users can generate QR codes, customize colors, change size, download, and share them.

---

## 🚀 Features

- Generate QR Code instantly
- Customize foreground and background colors
- Change QR size dynamically
- Download QR as image
- Share QR using Web Share API
- Debounced input for better performance
